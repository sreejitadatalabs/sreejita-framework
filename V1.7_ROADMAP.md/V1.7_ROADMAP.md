### Example 2: A/B Test Analysis
```python
from sreejita.stats import ABTest

# Analyze experiment results
test = ABTest(
    control_group=control_conversions,
    variant_group=variant_conversions,
    sample_size_control=10000,
    sample_size_variant=10000
)

results = test.calculate(
    power=0.8,
    significance=0.05
)

print(f"Winner: {results['winner']}")
print(f"Confidence: {results['confidence']:.1%}")
```
