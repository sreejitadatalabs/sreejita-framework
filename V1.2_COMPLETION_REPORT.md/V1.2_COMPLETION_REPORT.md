## Core Module Assessment (Cleaner, Insights, KPIs)

### 1. Data Cleaner (`sreejita/core/cleaner.py`)

**Purpose**: Standardize and clean raw dataframes

**Functions**:
- `clean_dataframe(df: pd.DataFrame, preserve_date_cols: list = None)` → Returns dict with cleaned 'df' and 'summary'

**Capabilities**:
✅ Column name standardization (lowercase, underscore-separated)
✅ Duplicate removal
✅ Whitespace normalization in text columns
✅ Empty string to NaN conversion
✅ Index reset
✅ Comprehensive cleaning summary (rows, columns, dtypes)

**Status**: **READY FOR PRODUCTION**

---

### 2. Insights Generator (`sreejita/core/insights.py`)

**Purpose**: Extract statistical insights from cleaned data

**Functions**:
- `correlation_insights(df, target="sales")` → List of insight strings

**Capabilities**:
✅ Correlation analysis with target variable
✅ Identifies strongest correlations by magnitude
✅ Generates human-readable insight strings
✅ Handles missing target columns gracefully
✅ Numeric-only column filtering

**Status**: **READY FOR PRODUCTION**

---

### 3. KPI Computation (`sreejita/core/kpis.py`)

**Purpose**: Compute key performance indicators

**Functions**:
- `compute_kpis(df, base_kpis=None, domain_kpis=None)` → Dict of KPIs

**Capabilities**:
✅ Base KPIs: Row and column counts
✅ Extensible base_kpis parameter for custom metrics
✅ Domain-specific KPI merging
✅ Flexible architecture for domain customization

**Status**: **READY FOR PRODUCTION**
